# NeedMed

NeedMed is a modular MERN omni-channel pharmacy platform with customer, admin, rider, and consultation workflows.

## Workspace

- `server`: Express and MongoDB API
- `apps/customer`: customer storefront
- `apps/admin`: pharmacist and admin dashboard
- `apps/rider`: rider portal

## Quick start

1. Copy `.env.example` to `.env` in the root and `server/.env.example` to `server/.env`.
2. Install dependencies from the repo root.
3. Start the API and frontend apps.

## Current status

This repository contains the initial production-oriented scaffold. Core business modules will be added incrementally on top of this structure.
